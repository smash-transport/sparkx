SPARKX

Software Package for Analyzing Relativistic Kinematics in Collision eXperiments

This package is designed to assist users of the SMASH and JETSCAPE/X-SCAPE codes to analyze the output of the simulation data. It contains multiple classes to read in the simulation outputs in OSCAR2013/OSCAR2013Extended format or the hadron output of JETSCAPE/X-SCAPE.
