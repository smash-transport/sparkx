SPARKX

Software Package for Analyzing Relativistic Kinematics in Collision eXperiments

This package is designed to assist users of the SMASH and JETSCAPE/X-SCAPE codes to analyze the output of the simulation data. It contains multiple classes to read in the simulation outputs in Oscar2013/Oscar2013Extended/Oscar2013Extended_IC/Oscar2013Extended_Photons/ASCII format or the hadron/parton output of JETSCAPE/X-SCAPE.
